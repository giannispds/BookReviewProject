-- Λογαριασμοί Χρηστών (Login/Register(Idetity))

Η εφαρμογή περιλαμβάνει έτοιμους 3 απλούς χρήστες για δοκιμές.
Μπορείτε να συνδεθείτε με οποιονδήποτε από αυτούς:

Username: giannis — Password: Giannis1! — Email: giannis@test.com

Username: alex — Password: Alex1! — Email: alex@test.com

Username: jane — Password: Jane1! — Email: jane@test.com

Με αυτούς τους λογαριασμούς μπορείτε:

Να δείτε όλα τα βιβλία

Να αφήσετε κριτικές (reviews) σε βιβλία

Να κάνετε like/dislike σε ένα Review

Και τέλος υπάρχει και Register για εγγραφή νέου Χρήστη.



-- Admin Panel

Υπάρχει ξεχωριστός λογαριασμός διαχειριστή (Admin) που έχει δικαιώματα για όλα.

Email / Username: admin@demo.local

Password: Admin#12345!

Μόνο αυτός ο χρήστης έχει πρόσβαση στο Admin Panel.
Μέσα από εκεί μπορεί να:

Δημιουργεί νέα βιβλία

Ενημερώνει υπάρχοντα βιβλία



-- Books

Τα βιβλία μπορούν να δημιουργηθούν/τροποποιηθούν μόνο από τον Admin.

Οι απλοί χρήστες δεν έχουν πρόσβαση στη δημιουργία ή επεξεργασία βιβλίων.


-- Κριτικές (Reviews)

Μόνο οι συνδεδεμένοι χρήστες μπορούν να προσθέσουν κριτική.

Κάθε review περιλαμβάνει: Βαθμολογία (1–5)

Σχόλιο (προαιρετικό κείμενο)

Όνομα του χρήστη που το έγραψε

Οι επισκέπτες που δεν έχουν συνδεθεί βλέπουν τις κριτικές, αλλά δεν μπορούν να υποβάλουν νέα.



-- REST API (Swagger)

Εκτός από το UI σε Razor, έχουν υλοποιηθεί και REST APIs:

BookApiController → CRUD για βιβλία

ReviewApiController → CRUD για κριτικές

Αν τρέξετε την εφαρμογή και πάτε στο:

-- https://localhost:7259/swagger/index.html

θα δείτε το Swagger UI όπου μπορείτε να δοκιμάσετε όλα τα endpoints.



**"Η εφαρμογή χρησιμοποιεί SQLite ως βάση δεδομένων και όλα τα δεδομένα (χρήστες, βιβλία, reviews) αποθηκεύονται τοπικά."
